Welcome

Welcome to Think Smart with C#. This is a tutorial website that I have designed to allow people learn the basics of coding using the C# language and the .NET framework.

C# is an excellent first language to learn because of it's easy syntax and versatility. Throughout this small tutorial we will learn about data types, conditional statement, loops, methods and classes.

This is only an introduction to coding in general but my hope is that by following along you will understand the basics and realize that not only is coding not as difficult as some make it out to be, but it is incredibly fun too!
