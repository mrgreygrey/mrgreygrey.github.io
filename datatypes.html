What is a data type?

Data type is an attribute we assign to data so that the compiler knows what to do with it. Knowing the different data types is critical to your ability to write good code. While not all languages force you to do so, assigning type to data is important in the C# language.

C# is what is known as a stronly typed language, which means that every time a variable is declared the type must be assigned to it. The most common types you will declare are:

Type Declaration Description
Integer int A 32 bit integer that ranges from-2,147,483,648 to 2,147,483,647
Double double A 64 bit number with a floating decimal point that ranges from (+/-)5.0 x 10-324 to (+/-)1.7 x 10308
Boolean Boolean A data type that is simply true or false
Char char A single alpha-numeric character
String String the most common object; a string is any string of characters

This is only a small number of the data types available to you. However, most other data types you will deal with are variations of these five. Some are larger some are smaller and as you learn more about how programming works you will know which one is best for any number of situations.
