IDEs

Visual Studios

An excellent IDE developed by Microsoft spefically for use with c# and the .NET framework. Can also be used with several other languages. Is heavily supported by Microsoft and some third parties. Can also be used with several other languages. Visual Studio Download

Visual Studio Code

A lightweight IDE that works on all desktop operating systems. Quick and easy to set up, It also has a robust third party support network to go along with the support from Microsoft. Like Visual Studios VS Code also includes Intellisense predictive typing for quick and easy coding. VS Code Download

C# API

The .NET API is an excellent resource for .NET developers. A long with detailed explanations of all classes and associated methods used in the .NET framework it contains lots of example code to help you understand the most effective ways to use the C# language. Microsoft .NET API
