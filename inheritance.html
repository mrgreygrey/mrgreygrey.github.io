What is inheritance?

Inheritance is the idea that one class can inherit attributes from parent class. This is one of the most valuable parts of object-oriented programming as it allows for a remarkable amount of code reusability.

The picture above shows a great example of how this works. At the highest level all of those objects are infact animals. That means that they all share similar traits such as having a heart lungs etc. and they can all perform different actions(breathing, moving). And they are all mammals which also means that they share even more similar traits. further down they start to diverge but they all retain the traits they inherited from the parent classes.

By having all classes with similar traits inherit from a parent class we don't have to write the same code over and over again. And we can just use the data and methods inherited whenever needed.
