What is scope?

Scope is what determines which data can be seen by what part of the program. This is determined mainly by two things within the C# language: visibility declaration and curly braces.

Access Modifiers

Access Modifiers are something you will see all the time when you start writing code. They are used to tell the compiler what data can be seen where. It is important that we don't allow data to be seen by any part of the program that doesn't doesn't have and inherent need to see it. the four main access modifiers you will deal with are:

Name Description
Public Can be accessed by any part of the program
Private Can only be accessed by the class it is in
Protected Can only be accessed by this class or a class derived from it
Internal Can be accessed by any member in the same assembly

Curly Braces { }

Curly braces are how we determine where a body of code starts and finishes. This includes but is not limited to classes, methods and arrays. So when we create a class or write a method all code that belongs to it must be between those curly braces.

Main Method

What is the main method? Well simply put it is the point of entry for the program. When a program is executed this is where it starts. The syntax for the main method is different from language to language but in C# it looks like this:

public static void main(String[] args){
}
