The tool of the trade

The most common way to develop software in the modern era is with a program known as an Integrated Develop Environment or IDE for short. In the resource section you will find links to the best IDEs for C# application development. I recommend you download one and experiment with writing code as you go through this tutorial.

Starting a new project

The quick way to start a project in Visual Studios is:
1.Select File=>New=>Project
2.Rename project however you wish
3.Press Ok

When you hit enter and Visual Studio finishes the build you will be greated with the starting code in your editor. This includes the main method which we will discuss more further down.

The quick way to start a project in VS Code:
1.Select File=>New=>File
2.Rename file program.cs
3.Type the following into the code editor:
class program{
public static void main(string[] args){
Console.write("Hello World");
}
}

This will create your starting point which is where you can play with the examples.
